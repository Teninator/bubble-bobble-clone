# Cavern — Refactored PyGame Zero Game

This project is a refactored version of the PyGame Zero *Cavern* (Bubble Bobble–style) game from *Code the Classics*.  
The refactor improves structure and maintainability without altering gameplay.

---

## Requirements
- Python 3.5+
- pgzero 1.2+
- pygame

Install dependencies:
```bash
python -m pgzrun src/main.py
